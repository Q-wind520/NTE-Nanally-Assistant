## 简述

NTE Nanally Assistant（异环娜娜莉助手）是一个基于计算机视觉、模拟键鼠操作的脚本项目，可以帮助你在异环中自动完成一些枯燥的任务。

## 快速开始

### 推荐使用方式

从release下载最新的setup.exe，安装后以管理员身份运行。

### 从源码打开

本项目的代码目录为/src，入口函数/src/main.py，依赖库/src/requirements.txt，模板资源/assets，脚本目录/src/core/scripts。
克隆仓库后搭建虚拟环境（python version 3.12.8），安装依赖，运行main.py即可，可通过脚本目录自编写脚本。

### 注意

分辨率支持： ~~仅支持1080P分辨率游戏窗口。~~ 现已支持所有16:9分辨率窗口ψ(｀∇´)ψ，模板基于720P，因此游戏窗口在1280×720下脚本性能最佳。    
扩展屏支持：多屏幕下存在未知原因Bug，目前建议在单屏幕使用。
