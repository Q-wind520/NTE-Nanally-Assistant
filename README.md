# NTE Nanally Assistant(异环_NA)
---
## 简述
NTE Nanally Assistant（异环娜娜莉助手）是一个基于PyAutoGUI、针对《异环》的脚本项目，可以帮助你在异环中自动完成一些枯燥的任务。
