## 简述

NTE Nanally Assistant（异环娜娜莉助手）是一个基于计算机视觉、模拟键鼠操作的脚本项目，可以帮助你在异环中自动完成一些枯燥的任务。

## 快速开始

从release下载最新的版本，管理员权限打开，输入需要执行的脚本及次数即可。

### 注意

分辨率支持：由于未经更多测试，仅支持1080P分辨率游戏窗口，最好在16:9屏幕使用。    
多屏幕支持：多屏下可能存在未知原因Bug，目前建议在单屏幕使用。
